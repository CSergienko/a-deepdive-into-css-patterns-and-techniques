Types of Selectors

How you decide which elements to target.

Element Selectors

h1 {
  color: red;
}

h1, h2, h3, h4, h5, h6 {
  font-weight: 900;
  margin: 1.5em 0 1em;
}

p, blockquote {
  line-height: 1.2;
}

Pseudo-Elements

blockquote::before {
  content: open-quote;
}

blockquote::after {
  content: close-quote;
}

Class Selectors

.foo {
  color: red;
}

.foo:hover,
button.foo:focus {
  color: green;
}

Pseudo-Classes

button:hover,
button:active,
button:focus {
  color: red;
}

table tr:nth-child(2n) {
  background: #f3f3f3;
}

[type="radio"]:checked {
  transform: scale(5);
}

ID Selectors

#foo {
  color: red;
}

#foo:hover,
button#foo:focus {
  color: green;
}

Attribute Selectors


                
// Just the attribute
[class],
[id],
[data-mycustomattr],
[title] {
  color: red;
}

// Exact value match
[class="bar"],
[id="foo"],
[data-mycustomattr="fizzbuzz"],
[title="I'm a button, yo."] {
  color: red;
}

// Value anywhere in string (even as a substring)
[class*="baz"] {
  color: red;
}

// Exact value, or followed by a -
[class|="baz"] {
  color: red;
}

The Wildcard/Universal Selector

*,
*::before,
*::after {
  box-sizing: border-box;
}

Combinators

// Descendant
h1 small {
  font-size: 25%;
}

// Child
ul > li {
  line-height: 1.2;
}

// Adjacent Sibling
li + li {
  margin-top: 10px;
}

// General Sibling
p ~ span {
  color: gold;
}
